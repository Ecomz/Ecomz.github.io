What is this?

Ecomz is an open source project developed to provide simple, fast and connectables bundles to supporte ecommerce needs.

The Bundles design avoid business rules dependecy to turn each one usefull in more than one workflow. Beside this, each bundle support business needs through access to important data and format transformations.

This is the list of Needs I plan a bundle:

  • Payment
  • Cart
  • Checkout
  • User Profile
  • Fraud/Risk
  • Campaign
  • REST API
